   1. Установка сервера kRPC для KSP
1. Перейди на [официальную страницу kRPC на GitHub](https://github.com/krpc/krpc/releases).
2. скачай файл с расширением **.zip** (например, **kRPC-x.x.x.zip**, где "x.x.x" — это версия).

   2. Установка kRPC в папку с KSP
1. Открой папку, куда ты распаковал архив.
2. Внутри этой папки найдешь папку **GameData**.
3. Скопируй эту папку **GameData** в папку, где установлен **Kerbal Space Program**. Путь будет примерно таким:
   Убедись, что в папке `Kerbal Space Program\GameData` теперь есть папка **kRPC**.

   3. Установка библиотеки kRPC для Python
Теперь нужно установить клиентскую библиотеку для Python, чтобы твои скрипты могли подключаться к серверу kRPC.

1. Открой **командную строку** (можно через поиск в меню "Пуск" — просто набери `cmd`).
2. В командной строке введите команду для установки через **pip**:
   ```bash
   pip install krpc
   ```
   
   4. Запусти сервер kRPC в KSP
1. Запусти **Kerbal Space Program**.
2. В игре зайди в меню **Addons** и выбери **kRPC**.
3. Нажми на **Start Server** (по умолчанию порт будет `50000`, а IP — `127.0.0.1`).
4. Ты можешь также установить пароль для доступа, если хочешь защитить сервер.
